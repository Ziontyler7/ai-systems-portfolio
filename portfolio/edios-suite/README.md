# Edios Suite

## Overview

The Edios Suite is a six-application iOS portfolio created and developed by Michael Tyler.

The underlying Edios initiative began in 2024. **Edios Digital Memory Platform LLC** was established in March 2026 as a private-first consumer software incubator and digital experience lab built around absolute digital ownership, local-first operation, and secure user-controlled data.

Michael's role across the suite includes product strategy, systems architecture, product engineering, interaction design, technical documentation, evidence development, and release preparation.

## Applications

1. **Radius** — creator commerce and local-business platform
2. **Comedy Camera** — entertainment and performance platform
3. **Amateur Artist** — art discovery and gallery platform
4. **MindReader** — social guessing game
5. **Ennerstellar** — memorial technology and space-oriented remembrance experience
6. **Edios Digital Memory Platform** — private-first digital memory application

## Release status

All six applications have reached TestFlight distribution for beta testing.

This is a controlled beta-distribution claim. It is not an App Store launch claim, a public-user-scale claim, or a commercial-performance claim.
