# Public Boundaries

This repository is public by design.

It includes professional evidence, public-safe technical writing, architecture summaries, and author-facing context.

It does not include:

- private source code
- private prompts
- API credentials
- internal memory systems
- private operational logs
- protected implementation details
- unreleased product strategy
- sensitive personal information
- private financial context

The goal is to demonstrate capability without exposing protected systems.
